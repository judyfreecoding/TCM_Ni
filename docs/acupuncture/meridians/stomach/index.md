# 足阳明胃经（AM 7-9）

::: info 阳经的穴性
井 金

荥 水

俞 木

经 火

合 土
:::

足阳明胃经从头顶一路到脚趾，很大的一条经。有45个穴位。厉兑、内庭、陷谷、冲阳、解溪、足三里，是井荥俞原经合。
正常人，一息四至，也就是一吸，心脏跳两下，一呼，心脏跳两下。胃的蠕动和心跳一致，一吸，胃蠕动两下，一呼，胃蠕动两下。
胃的脉象，在右手的关部体现。

::: info 脉象基础知识
右手脉，寸，肺大肠；关，脾胃；尺，肾和子宫
左手脉，寸，心小肠；关，肝胆；尺，肾和膀胱
实脉，按得越重跳得越实在；虚脉，一按重就没了。
:::

## 气户

气户和缺盆，隔了一个锁骨。

::: info 胃经路线
从气户开始，直下到鹰窗穿过乳中到乳根，这一条线，是胃经循行。这条线距离任脉是4寸。从乳根往下，胃经路线与任脉的距离是2寸。
![An image](/stomach/9胃经在胸腹的线路.png)
:::

::: warning 肋间下针方法
肋间下针，无论是胸部还是背部，都一样，都是斜刺，从中间往两边斜刺。从穴位旁5分下针，斜向穴位进针，确保针尖刚好在穴位上。
:::

## 鹰窗乳中乳根

胃经经过乳中。乳中也是肝经的出口。

::: tip 产妇初次喂母乳
产妇初次喂母乳，如果小孩吸不出奶，用葱白涂乳头，一刺激，奶就出来了，效果很好。
:::

::: info 乳岩有古书记载
乳岩，硬如石，不痛，十几年过后，开始发臭，腐尸的臭味。
:::

::: info 奶水，月事也
奶水，月事也。是心在管。所以月经前两天，乳房会胀满，那就是奶水胀满了。月经和排卵，是两码事。停经以后，照样有奶水。所以停经以后，只能通过大便排出来。经前每天大便一次，停经后就需要每天大便两次。
:::

::: tip 乳房硬块
乳房硬块，也就是乳癌早期，天应隔姜灸，或循经取穴，子母穴、郄穴下针，都可以。效果都很好，一两次就会变软变小。药都不需要用。远取穴必须要把经找准，硬块长在胃经上，远取穴取在脾经、心经上，是没有用的。所以认准经络路线是非常必要的。
:::

## 不容承满

从不容开始，胃经和任脉距离2寸。

穴位一定是在缝隙中，这里一路是腹肌，所以胃经是沿着腹肌走的，每个人腹肌大小有不同，所以穴位也不是一定在2寸旁开的位置，但一定在腹肌旁边的缝隙中。

![An image](/stomach/10不容.png)
不容，巨阙旁开2寸。就是食物不容。当食物不消化的时候，用不容穴。比如心口痛喝水就吐，下巨阙和不容。所以不容实际上是巨阙的辅助穴。

上脘旁开2寸，是承满。承满的意思就是，吃一点点就满了，很容易胀满，这时就可以用承满穴。

## 关门太乙滑肉门

![An image](/stomach/11关门太乙滑肉门天枢.png)

任脉的建里下脘水分的旁开2寸，分别是它们的辅助穴，分别叫做关门、太乙、滑肉门。

::: info 增高
太乙滑肉门，用于增高。身柱、太乙、滑肉门，是三个增高的穴位。
:::

## 天枢（大肠募）

![An image](/stomach/11天枢.png)
神阙旁开2寸。腹泻，灸神阙，灸后如果还有一点肠鸣腹泻，就配合针天枢。

天枢是大肠募穴。

::: tip 便秘的诊断和治疗：天枢
天枢是帮助诊断便秘的穴位。天枢压痛惨叫，就下针，帮助病人排便，直到天枢没有压痛了，压下去也是软软的了，就说明宿便排掉了。
:::

::: info 便秘与精神病的关系
便秘，要确定是堵在大肠、小肠、十二指肠、还是胃里。天枢压痛惨叫，说明便秘在大肠，是燥矢、宿便。如果便秘太久，燥矢的浊气已经浓得横膈膜挡不住了，浊气就会上升到心肺腔，然后人就发狂奔走了，这就是精神病登高而歌裸奔于市的原由。所以精神病院里的病人十有九都是便秘，还有那一个没便秘的就是假的。
:::

::: tip 便秘的诊断和治疗：天枢 + 神阙
便秘分寒、热。寒，是因为大肠不蠕动，所以不排便，也不会痛；热，是大肠在蠕动，但是大便太干排不出来，所以病人会痛。有，就是实。所以病人大便排不出来肚子痛，就称为热实。病人大便排不出来但肚子不痛，就称为寒实。

如果是寒症，灸比针好，灸天枢；隔姜灸。可以同时隔盐灸神阙，效果更好。盐，要用炒过的盐。

如果是热实，针比灸好，针天枢。
:::

::: tip 便秘的治疗：腹四针（中脘关元天枢）
如果一个病人胃口不好，便秘，肚子痛，小便却淡白。

注意，便秘+肚子痛，是热实，一般会小便褐黄，但这个病人小便淡白。说明他热实夹寒。

这种情况用腹四针：中脘、关元、天枢。

小便淡白的时候，灸天枢，针中脘关元；

小便褐黄的时候，全部下针。

腹四针，是大肠、小肠、胃一起治。下针后，捻转引到气就好了，不用补泻。
![An image](/stomach/12腹四针.png)
注：倪大师随手画的图，都是比例很精准的，这个图中肚脐上4是中脘，下3是关元，看着下3比上4更长，似乎画得不准。其实不是倪师画得不准，腹部的寸就是这样定义的。实际人身上的这几个穴位，就是这样的比例。
:::

## 梁丘（郄穴）

![An image](/stomach/15阴市梁丘解穴.png)

## 足三里（合穴，属土，本穴）

![An image](/stomach/17足三里.png)
足三里，膝盖骨下3寸。快捷找法，顺胫骨往上，摸到有高骨刚刚出现，再外一横指，就是足三里。也就是说，足三里，在胫骨的阳侧。

胃经大穴，治一切胃病，虚实都治。胃经合穴，属土，本穴。

::: info 常灸足三里
年过三十，就要多灸足三里，会让胃气很好，就能健康长寿。可灸百壮。如果上身有火，可灸足三里，把火气往下导。
:::

::: tip 乳房硬块
乳房硬块，正好长在乳中的上方或下方，这就是在胃经上，足三里好用。超过5天，行补法；如何硬块压着很痛，行泻法。
:::

::: warning 足三里的补泻针法
胃经经气是在阳面从上往下行，顺经气快转慢回，是补；逆经气快转慢回，是泻。
![An image](/stomach/18足三里补泻法.png)
如果是左腿，逆时针为泻，顺时针为补；
如果是右腿，逆时针为补，顺时针为泻。
所以双手在左右腿上同时行补或行泻，是相同的手法，很容易同时完成。行3、5、7下补法以后，再提豆许，这样补气是很强的。
:::

::: tip 用足三里治疗血压上升
病人血压上升的时候。中医说是阳亢，脸充血；西医说是血压高。在足三里下五分下针，斜刺到足三里，阳明热慢慢就泻下来了。

这种刺法是用的泻的手法。
:::

::: info 中医对疾病和治病的看法
- 中医的治病观念。中医认为，有胃气则生，无胃气则死。无论肝癌肺癌还是感冒等等，处方下去，如果胃口恢复，说明胃气回来了，就代表救回来了。因为胃气回来，说明人体系统开始自主运行了，自己产能，自己运转，自己排毒，所以一旦开始自主运行，就能自己处理外邪内邪，身体慢慢就会好。所以中医认为恢复胃气就叫救回来了。如果一直治却一直都没有恢复胃口，就表示治错了。

- 中医对疾病的定义。中医认为平衡为正，不平衡为病。所谓不平衡，就是病人自己觉得不舒服不自在；所谓平衡，就是人正常的状态。人正常的状态是，每天能吃能喝能拉能睡且没有感觉不舒服。身体系统自己会找平衡，可能有癌细胞，但只要身体能保持平衡，癌细胞就不会作乱。所以不要用X光等外力去刺激它让它从沉睡中醒来作乱。澳洲南方有个小渔村，是澳洲最长寿的地方，渔民靠捕鱼维生，大多活到一百多岁，九十多岁的人看起来像中年人，身体很好。解剖长寿者的尸体，发现很多人都有西医定义中的肝癌肺癌等等，但他们都不是死于癌症，而是因一百一二十岁组织器官老化退化而自然去世的。研究他们的食物，发现多为海带海藻类。

- 中医治病关注的就是治症。本来没有胃口，治成有胃口了；本来睡不好，治成睡得好了；本来全身黄疸，治后黄疸退了；本来小便排不出，治后小便通畅了；本来水肿，治后水肿消了。全身也没有哪里不舒服了，这就可以了。不必非要去检查癌细胞还在不在，或者非要去关心那些西医的指标。
:::

::: info 海洋食物对人类有益
人类来自海洋，所以海洋食物对人类有特殊意义。海带海藻，黑，咸，入肾。
:::

## 阑尾穴（奇穴）

阑尾穴，足三里下一寸。是诊断和治疗阑尾炎的奇穴。

诊断：如果腹痛，阑尾穴有压痛（男左女右），说明是阑尾炎。

治疗：阑尾穴最痛点下一针，引到气就好，腹痛当场去掉。配天枢、关元等，帮助病人排便。

上巨虚，足三里下3寸。上面有巨使之虚，治胃经上的乳房硬块等。

## 丰隆

![An image](/stomach/20丰隆.png)
丰隆，小腿外侧，腓骨上下两高点的正中间。丰胸效果好，用补法，不要用泻法，不然正好效果相反。袪痰导痰效果好。癫癎病人十之有九有痰，呼吸时都能听到痰音。

::: warning 丰隆导痰针法
丰隆直下，捻转引到气，然后提针到皮层，针头朝上45度进针，再捻转引到气，又提针到皮层，针头朝下45度进针，下去以后，针就留在那边二十分钟。
:::

::: info 针灸疗程问题
- 对于新病，很多扎一次就好。比如经络痛、扭伤、落枕、五十肩、不能扭毛巾等。

- 也有新病，需要多扎几次，一般是每日扎一次，连扎几日。比如倪师治面部中风口眼㖞斜。

- 也有几十年老病，穴位用对了，时机用对了，也是一两次就好。比如倪师治偏头痛。几十年老病的，一定是在发病时治。

- 对于老病或需慢慢长好的问题，比如丰胸，就是按疗程治，一般隔天扎一次。
:::

冲阳，大约在解溪下两寸，摸到动脉下方就是冲阳，拨开动脉下针。原穴。

::: info 为什么胃经穴位可治精神病？
书上写的胃经穴位，很多都治“发狂奔走、登高而歌、弃衣而走”，这是因为肠胃有病，致大便不通，燥矢气上冲，让人头脑狂乱。所以胃经的穴位能治精神病。
:::

## 陷谷（俞穴，属木）

![An image](/stomach/22厉兑内庭陷谷.png)
陷谷，内庭上2寸。俞穴，属木。

::: tip 胃经时症用陷谷
上午7-9点发的病，不管是啥病，都下陷谷，比如每天早上脸肿、每天早上发火、每天早上头痛。
:::

另外，退奶可以下陷谷、上巨虚。

## 内庭（荥穴，属水）

内庭，二三趾间后约五分，骨前一点。荥穴，属水。下内庭，一定是要治实症，一定是逆经下针，即朝着身体方向进针；进针后逆经快转慢回；行4次6次泻针后再推豆许。一次下针，行所有泻法，泻的能力很强。治对侧下牙痛。如果下牙化脓发炎，下梁丘。

::: info 上下牙的线路
大肠经走上牙。上牙痛是合谷。如果上牙化脓发炎，下温溜。

胃经走下牙。下牙痛是内庭。如果下牙化脓发炎，下梁丘。

采对侧治疗法。门牙痛就两边一起扎。

近取穴也可以下下关、颊车等。
:::

::: info
痛，就是堵，就是有实，就用泻，用子穴，或者用原穴，行泻的手法。
:::

## 厉兑（井穴，属金，子穴）

厉兑，二趾甲角（朝三趾方向）。井穴，属金，子穴。治实症。厉兑穴治心脏实症很厉害，比如心脏动脉血管堵塞剧痛，一下针，病人会感觉很舒服，感到一个小扫把在扫心脏。如果需要行泻手法，泻荥穴内庭。

::: warning 井穴的补泻法
井穴没有办法行补泻，而且扎井穴很痛，如果井穴恰好是母穴或子穴，不得不扎，那么：

**补井当补合，泻井当泻荥**

意思就是，如果井穴是母穴，那么就把合穴当母穴来用；

如果井穴是子穴，那么就把荥穴当子穴来用。

所以胃经的荥穴内庭，就是当子穴在用。
:::

::: tip 井主心下满
心下，就是巨阙的位置，所以是心脏病。

井主心下满，意思就是厉兑治心脏病。厉兑治心脏病，效果是非常好的。
:::

::: tip 井穴治尸阙
尸阙，就是突然晕倒昏迷。

十宣放血，急救尸阙。
:::

::: info 山陵沟谷的应用
善针者，善导引也。山陵沟谷，都是导引穴。穴意都如其名。山陵之间都有沟有谷，让水顺着山陵沟谷一路导下去，就顺畅了。

比如水沟，病人阳亢昏迷时，一下水沟，就把阳亢导入水沟了。还有承山、阳陵、阴陵，手上的阳谷、支沟。

也就是说，有实的时候，无论是有瘀血，有积块，有症瘕，有积水，有阳亢，统统可以下所在经的山陵沟谷穴。

比如退奶。奶水，是胃经在管，所以下胃经的陷谷。退奶是泻嘛，所以胃经的子穴也可以。
:::
