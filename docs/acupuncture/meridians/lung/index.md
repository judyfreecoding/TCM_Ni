# 手太阴肺经（金）

## 肺是传导之官，治节出焉

治，治理；节，规律。像大自然一样，人体运行是有规律的，人体的规律是由肺在管。肺感应外面大自然的运行，并调节人体内部的运行使之与其一致。所以我们的日常生命活动要顺应大自然的规律。中国农历就是根据大自然的运行规律制定的，其中的节气就是指导人体活动的最佳指南。比如四立四分日，即立春立夏立秋立冬、春分夏至秋分冬至，这八个日子，是大自然变化天气的转折点，最不适宜人体活动，除了正常上班上学，这八天特别不适宜安排出游、出行、娱乐、体育等等活动，如果活动了，特别容易生病。

## 肺藏魄

魄，就是白天的鬼。人身上有七魄，这七魄是喜欢白天的。如果一个人不喜欢白天，喜欢黑暗，就是失魄了。

人身上的七魄，是抵御外界各种鬼邪的，就是以鬼治鬼的原则。人的七魄强，则抵抗力强，不易中邪感染细菌病毒而致生病。

## 肺是气之本

气，指热气，也就是蒸汽。蒸汽上升，所以头油就是蒸汽的表现，是肺气足的表现。

## 肺主皮毛

眉毛是肺在管。眉尾长长，是肺气足的表现，所以老年人眉尾长是长寿的标志。

肺管全身的皮肤。皮肤病，从肺治。

肺在肋骨里面，如何按摩肺？肺主皮毛，肺主气，手在皮肤表面轻轻摩擦，就动到气了，也就按摩到肺了。一定要轻，轻才能引动气。

## 肺开窍于鼻

除了鼻子，全身皮肤都在呼吸。

## 肺主忧

肺主忧，忧伤肺。人体需要淡淡的忧愁，但不能过忧。咳嗽时如果有担忧的事，咳嗽就不会好。

## 肺音商，发“呬”音，在声为哭

肺音商。音用于五脏逼毒法。发“呬”的音。牙闭紧，唇裂开，舌顶住门牙，发出“呬”的音。手尽量往上举，脚踮起，吐气吐完，这时才开始，提肛，然后吐出最后的气，发出“呬”的音。这是肺脏逼毒法。一天可以做五、六次。逼完毒后，再开始练功夫，会事半功倍。

## 时辰

肺经流注时间为寅时，即早上3-5点，从肺左流注到肺右。所以听到病人陈述“我总是凌晨3:30左右醒”，就知道他的病在左肺中部。

肺法象天幕。也就是肺就像天一样。天是冷的，所以皮肤是冷的。正常人，头面皮肤是冷的，手脚是温热的。

# 肺经穴位

## 云门中府（肺募）

![An image](/lung/2中府云门.png)

云门，肺之募。

中府，云门下一肋骨。

下针方法：下云门中府，都是一针透两穴。从中府下针，浮在肋骨上面，向上刺到云门，一针透两穴，这是顺经扎，为补。

## 天府侠白

天府，上臂内侧正中间的地方，也是手举起来鼻尖能靠到的地方。肘窝正中间有条大筋，大筋前侧是尺泽，从尺泽到腋横纹的正中间，在两个肌肉中间，就是天府。下针时，用手指把两个肌肉分开来，下针到肌肉缝中。专门治疗流鼻血，左鼻孔流血扎右边，右鼻孔流血扎左边。如果扎对了地方，扎下去，就会看到鼻血慢慢缓了，5分钟内鼻血就停了。

天府下一寸，是侠白。如果针天府还没有止到鼻血，加扎侠白。一寸距离的穴位，都是加强辅助作用的。

## 尺泽（合穴、属水、子穴）

![An image](/lung/3尺泽.png)

尺泽，肘窝正中间有条大筋，大筋前侧是尺泽。肺经合穴，属水。因为肺是金，金生水，所以水穴是肺经子穴。肺家实症，在此穴泻。除了肺家病，经络病也可以在此穴泻。比如手臂痛，一看是肺经上，痛5天，实症，对侧尺泽下，就解决了。如果痛超过5天，虚症，下母穴。

::: tip 用时间判断虚实
初病（一周以内）为实，久病为虚。

一周以上，无论是三个月，还是三年，还是三十年，都是虚症。
:::

只要扎到尺泽，引到气，就是泻了，因为它的穴性就是水，水对于金来说就是泻。不需要专门行补泻手法。

## 孔最（郄穴）

![An image](/lung/4孔最.png)
孔最，尺泽下三寸。也是在两个肌肉的中间。是肺经郄穴，郄穴用于消炎镇痛。针灸都可以。

特治痔疮。痔疮病家的孔最处有压痛。可能是准3寸处痛，也可能是2.5寸处痛，因为痔疮长的地方有不同，在压痛处下针。

## 列缺（络穴）（管任脉）

![An image](/lung/5列缺经渠太渊.png)

列缺经渠太渊，就是摸脉寸关尺的地方，这三穴都不灸，因为有动脉。穴位都在骨头和动脉血管之间，用手指拨开动脉，一寸针，贴骨进针，下到穴位后，再放开手指。

::: tip 肺经和大肠经的走线
肺经在骨头的阴侧，大肠经在骨头的阳侧。
:::

列缺管任脉，肺经络穴，络到大肠经。列缺治任脉的所有病，比如任脉上的痛、痘疮等。

## 经渠（经穴、属金、本穴）

经渠，肺经经穴，属金，本穴。在本穴可行补泻手法。

## 太渊（俞穴、属土、母穴）（脉会太渊）

太渊，脉会太渊，肺经俞穴，属土，母穴。在这里一下针，对肺来说就是补。

脉有病在这里下，比如脉跳跳停停，心率不齐。肺家虚症、经络虚症，在这里治。

## 鱼际（荥穴、属火）

![An image](/lung/6鱼际.png)

鱼际，肺经荥穴，属火。时症在荥穴、俞穴治。

::: tip 时症
按时辰发的症状，叫时症。时症只论时辰，不论症状，在哪个时辰，就是哪个脏腑的病。肺的时辰是寅时，早晨3-5点定时发病，不论是醒来，还是咳喘，还是肚子痛，等等，都是肺的问题。

时辰病，在荥穴、俞穴治。肺的时辰病，就下鱼际太渊。
:::

## 少商（井穴、属木）

![An image](/lung/7少商.png)

少商，肺经井穴，属木。点刺放血，挤一两滴。也可下针。治扁桃腺发炎痛、发烧。

扁桃腺发炎痛，可以直接在扁桃腺处点刺放血。用三寸针，左三针，右三针，血一出来，会好很多。倪师给一个美国女孩治，十几年扁桃腺发复发炎，倪放了几次血才好。放完血以后，可以下列缺照海，喉咙就不痛了，然后少商再放点血。

::: tip 喉咙痛的治法
- 列缺。喉咙属任脉，所以列缺正治喉咙痛。

- 照海。照海管阴跷脉，阴跷脉直络到喉咙，所以照海也治喉咙痛。

- 少商。少商是肺经的井穴，井穴力量最大，喉咙属肺系统，所以少商也治喉咙痛。一般是点刺放血，也可扎针。

- 天应放血。如果是扁桃腺肿大发炎，就在扁桃腺处放血。
:::
